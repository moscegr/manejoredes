Conoce la materia

Introducción

El módulo de Manejo de redes tiene como finalidad que adquieras las habilidades y destrezas necesarias para integrar y manejar dispositivos de red, proporcionando un nivel de seguridad óptimo mediante la configuración e interconexión de los equipos requeridos para brindar acceso a los usuarios.

Aprenderás a identificar y resolver problemas que se presentan en el manejo de la red de una organización, preparándote para un entorno laboral exigente y dinámico.

Propósito del Módulo

"Implementar y configurar dispositivos de red, de acuerdo con las necesidades específicas de una organización, para elevar los niveles de administración, rendimiento y seguridad de las redes de área local cableadas e inalámbricas."

Plan de Estudios

Temario del Semestre

El contenido está estructurado en tres unidades de aprendizaje o parciales para desarrollar tus habilidades de forma progresiva.

Parcial 1

Implementación de dispositivos de red inalámbricos

30 horas
  • 1.1

    Acceso a recursos inalámbricos

    Configuración del acceso a los recursos de la red inalámbrica a través de las herramientas que proveen los dispositivos de red (Puntos de Acceso, SSID, Canales).

  • 1.2

    Seguridad Inalámbrica

    Configuración de parámetros de seguridad (Filtrado MAC, Encriptación, Firmware) para proteger la información y mitigar amenazas comunes.

Parcial 2

Implementación de dispositivos de ruteo y conmutación

74 horas
  • 2.1

    Servicios de conectividad

    Uso de comandos del Sistema Operativo Internetwork (IOS) mediante la CLI para la configuración global de routers y switches en una LAN Ethernet.

  • 2.2

    Servicios integrados

    Configuración de servicios como DHCP, NAT estática y dinámica en dispositivos ISR (Integrated Services Router) usando CLI o SDM.

Parcial 3

Administración de redes de áreas locales virtuales

40 horas
  • 3.1

    Creación de VLANs y Troncales

    Agrupación lógica de estaciones de trabajo, asignación de puertos y configuración de enlaces troncales IEEE 802.1Q.

  • 3.2

    Protocolo VTP

    Administración de dominios VTP, configuración de switches en modo Servidor, Cliente o Transparente para propagar las VLAN.

  • 3.3

    Enrutamiento inter-VLAN

    Establecimiento de comunicación entre VLANs distintas usando la técnica "Router-on-a-stick" y subinterfaces.
